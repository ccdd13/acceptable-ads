# acceptable-ads
Ad blocking filter that whitelists good ads. Unlike ABP Acceptable Ads, it is not for-profit.

## Currently Unblocked
* [DuckDuckGo Search Ads](https://duck.co/help/company/advertising-and-affiliates)
* [Ecosia Seach Ads](https://ecosia.zendesk.com/hc/en-us/articles/201531142-What-are-EcoAds-)
* [ReadTheDocs Ethical Ads](https://docs.readthedocs.io/en/latest/ethical-advertising.html)
* [Carbon Ads](https://carbonads.net)
* [Nativo](https://nativo.com)
* [BuySellAds](https://buysellads.com)
* [Connatix](https://connatix.com)
* [Flipp](https://flipp.com/home)
