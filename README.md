# acceptable-ads
Ad blocking filter that whitelists good ads. Unlike ABP Acceptable Ads, it is not for-profit.

## Currently Unblocked
* [DuckDuckGo Search Ads](duck.co/help/company/advertising-and-affiliates)
* [Ecosia Seach Ads](ecosia.zendesk.com/hc/en-us/articles/201531142-What-are-EcoAds-)
* [ReadTheDocs Ethical Ads](docs.readthedocs.io/en/latest/ethical-advertising.html)
* [Carbon Ads](carbonads.net)
* [Nativo](nativo.com)
* [BuySellAds](buysellads.com)
* [Connatix](connatix.com)
* [Flipp](flipp.com/home)
